# 3초 생존 게임 기획서

## 1. 게임 개요
### 컨셉
- **장르**: 캐주얼 액션, 미니게임 컬렉션
- **플랫폼**: 모바일 웹 (반응형)
- **플레이 타임**: 1회 3초, 세션당 1~5분
- **핵심 메커니즘**: 무작위 미션을 3초 안에 성공하면 생존, 실패하면 게임오버

### 게임 플로우
```
시작 → 미션 제시 (0.5초) → 플레이어 액션 (2.5초) → 성공/실패 판정 → 다음 미션 or 게임오버
```

## 2. 핵심 시스템

### 2.1 미션 시스템
각 미션은 다음 요소로 구성:
- **미션 텍스트**: 간결한 한글 지시문 (예: "빨간색 탭!")
- **미션 타입**: TAP, SWIPE_UP, SWIPE_DOWN, SWIPE_LEFT, SWIPE_RIGHT, LONG_PRESS, MULTI_TAP
- **난이도**: EASY, NORMAL, HARD
- **시각적 요소**: 색상, 아이콘, 애니메이션

### 2.2 미션 종류 (Phase 1)

#### 기본 미션
1. **색깔 탭하기**
   - "빨간색 탭!" / "파란색 탭!" / "노란색 탭!"
   - 화면에 여러 색의 원이 등장, 올바른 색 탭하기

2. **방향 스와이프**
   - "위로!" / "아래로!" / "왼쪽으로!" / "오른쪽으로!"
   - 화살표 방향으로 스와이프

3. **반대로 하기**
   - "반대 방향!" (화살표와 반대로)
   - "안 되는 색!" (표시된 색 제외하고 탭)

4. **빠르게 탭하기**
   - "5번 탭!" / "10번 탭!"
   - 연타 미션

5. **길게 누르기**
   - "길게 눌러!"
   - 1초 이상 롱프레스

6. **두 개 동시 탭**
   - "둘 다 탭!"
   - 두 개의 타겟을 동시에 터치

### 2.3 난이도 시스템
- **스코어 기반 난이도 증가**
  - 0~5점: 3초내 미션 타입 1개
  - 6~15점: 3초내 미션 타입 2개
  - 16~30점: 3초내 미션 타입 3개
  - 31점+: 3초내 미션 타입 4개

### 2.4 피드백 시스템
- **성공 시**: 
  - 효과음 (딩!)
  - 화면 흔들림 + 파티클 이펙트
  - +1 점수 표시
  
- **실패 시**:
  - 효과음 (삑-)
  - 화면 붉은 플래시
  - "실패!" 텍스트 + 최종 점수

## 3. 기술 스택
- **프레임워크**: Vue 3 (Composition API)
- **빌드 도구**: Vite
- **언어**: TypeScript
- **스타일**: CSS3 (Flexbox, Grid, Animation)
- **상태 관리**: Vue Reactivity (Ref, Computed)
